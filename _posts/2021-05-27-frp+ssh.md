##  + 配置ssh
###  1. SSH 分为 Client 和 Server
Ubuntu 默认自带SSH Client，端口号为22。如果没有，可通过sudo apt-get install openssh-client 来安装。如果作为Server端则需要安装SSH Server。
###  2. 安装SSH Server并启动
+ 查看SSH Server是否安装   
 'dpkg -l|grep ssh'
